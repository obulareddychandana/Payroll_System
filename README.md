# Payroll Management System

## 📌 Overview

This is a full-stack Payroll Management System developed using **Django REST Framework** for the backend and **React** for the frontend.
The system allows managing employees and generating payroll with automatic salary calculations.

---

## 🚀 Features

* Employee Management (Add, Update, Delete, Search)
* Payroll Generation
* Automatic Salary Calculation (HRA, PF, Tax)
* Clean and Responsive UI
* REST API Integration

---

## 🛠️ Tech Stack

* Frontend: React
* Backend: Django REST Framework
* Database: MySQL
